# ⚡ SyntaxVault

A modern, sleek, and fully functional code snippet manager and live web editor. Designed with a premium dark-neon glassmorphism UI, SyntaxVault allows developers to write, save, preview, and manage their HTML, CSS, and JavaScript snippets locally.

## ✨ Key Features

* **🎨 Advanced Syntax Highlighting:** Integrated with Prism.js for a beautiful, VS Code-like coding experience.
* **💾 Local Storage Persistence:** Never lose your work. All snippets are automatically saved to your browser's local storage.
* **▶️ Live Preview Engine:** Instantly test your HTML, CSS, and JavaScript code in an isolated iframe environment.
* **📥 Export & Download:** Turn your snippets into actual files! Download your code directly as `.html`, `.css`, or `.js` files with a single click.
* **📋 1-Click Copy:** Quickly copy your entire code block to the clipboard with a smooth visual confirmation.
* **🔍 Real-Time Search:** Easily filter and find your saved snippets using the built-in search bar.
* **📱 Fully Responsive:** Features a mobile-friendly layout with a slick Hamburger Menu, ensuring a great experience on desktops, tablets, and smartphones.

## 🛠️ Technologies Used

* **Frontend:** HTML5, CSS3 (Modern Flexbox/Grid, CSS Variables, Media Queries)
* **Logic:** Vanilla JavaScript (ES6+, DOM Manipulation, Blob API for downloads, Clipboard API)
* **Libraries/Assets:** * [Prism.js](https://prismjs.com/) (Syntax Highlighting)
  * [Boxicons](https://boxicons.com/) (Premium UI Icons)
